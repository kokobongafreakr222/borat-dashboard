# Borat Dashboard

Веб-дашборд для отслеживания проектов и метрик.

## Функции

- 📊 Статистика проектов (всего/активные/завершённые)
- 🔍 Фильтрация по статусу
- 🔗 Ссылки на GitHub
- 🏷️ Теги технологий
- 🕐 Дата последнего обновления

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой

Сборка статического сайта:
```bash
npm run build
```

Файлы в `dist/` можно задеплоить на GitHub Pages / Vercel / Netlify.
